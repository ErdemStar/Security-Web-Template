Hoşgeldiniz

Bugün sizlere Hackademic RTB1'in nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız hedefe sızıp root olmak.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile IP adresi tespiti
  • Nmap ile açık port'ların tespiti
  • Sqlmap ile database üzerinden bilgi alma
  • Wordpress'i kullanıp manuel olarak reverse-shell
  • Exploit-suggester ile hak yükseltme

İlk adımda netdiscover komutu ile hedefin kulladığı IP adresini tespit ettim.

Nmap kullanarak hedef üzerinde çalışan servis ve portları buldum.

Dirb kullanmadan önce 80 port'u üzerinde çalışan sayfası incelerken sql injection buldum.

Daha sonra sqlmap'ı kullanarak hedef üzerinde olan database'leri görüntülemek için kullandım.

Aşağıda hedef üzerindeki MySQL database'leri görebilirsiniz.

Daha sonra wordpress database'i üzerinde hangi bilgilerin tutulduğunu öğrenmek istedim.

Hedef üzerinde wordpress çalıştığını dirb kullanmadan MySQL üzerinde oluşturulan wordpress database ile öğrendim.Aşağıdaki fotoğrafta wordpress'e login olmak için elde ettiğim username ve password'ler var.

Bu işlemden sonra metasploit'i kullanarak hedeften kolayca shell alabilirdim fakat ben burada bir php-reverse-shell script'i ekleyip bu işlemi manuel olarak yapmak istedim.

Script'i upload ettikten sonra script'in hangi yolda olduğunu bulup tek yapmam gereken script'e tıklamak olacaktır.

Aşağıda görebileceğiniz gibi hedefe shell oldum.Bu işlemden sonra hedef üzerinde bilgi toplamak için tekrar linenum.sh script'ini kullandım

Hedef üzerinde exploit-suggester.sh script'ini kullanarak hedef'in kernel sürümüne göre exploit indirip doğrudan root olmaya çalıştım.İşaretlediğim exploit bu makinede root olmak için yeterlidir.

Root olduktan sonra flag'ımızı okuyup bu CTF'i başarı bir şekilde sonlandırıyorum :)
