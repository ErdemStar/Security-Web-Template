Hoşgeldiniz

Bugün sizlere Kioptrix Level 4'in nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız root olmak olacaktır.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile hedef'in ip adresini öğrenme
  • Nmap ile açık port'ların tespiti
  • Dirb ile 80 port üzerinde keşif yapma
  • Enum4linux ile 445 portu üzerinden bilgi toplama
  • SQL Login Bypass
  • MySQL üzerinden sistem komutu çalıştırma

Hedefin kullandığı ip adresini netdiscoveri ile bularak CTF'imize giriş yapıyoruz.

IP adresini öğrendikten sonra vakit kaybetmeden nmap ile açık portlar ve servisler hakkında bilgi topladım.

80 portunun açık olduğunu öğrendikten sonra bu port üzerinden bilgi toplamak için dirb'i kullandım.

Nmap çıktısından 445 portunda samba çalıştığını öğrendim.Bu port üzerinden bilgi toplamak için enum4linux tool'unu kulladındım.

Enum4linux ile birkaç username bilgisi elde ettim.Bunları tool'un çıktısından kopyalayıp cut ile sade bir hale getirdim.

Dahas sonra dirb'den elde ettiğim pathleri url'de tek tek denedim ve fotoğraftaki gibi bir login ekranınıyla karşılaştım.

Enum4linux'den elde ettiğim username'leri SQL Injection payloadlarla kullanarak login olmaya çalıştım.Fotoğraftaki gibi bir çıktı aldım.

Username bilgisi girdikten sonra ekranda gelen diğer bilgileri password olarak düşünüp SSH login olmayı deneyip sonuç aldım.Daha sonra beni limited shell karşıladı.Birkaç komut hariç hiçbirşey yapamıyordum.Bu adımda "echo" ile "bash"'e geçip atlattım.

PHP script'lerinin çalıştığı path'leri karıştırdığımda mysql server'a root olarak bir password vermeden login olabileceğimi öğrendim.

MySQL üzerinde birkaç araştırma yaptığımda "sys_exec" adlı bir function'nun olduğunu ve hedef üzerinde komut çalıştırabildiğimi öğrendim.Burada john'u admin gruba ekledim.Daha sonrada john'un password ile "sudo su" komutunu kullanıp root oldum.

"/root" path'ine giderek flag.txt'yi okudum.Bu adımdan sonra bu CTF'i başarıyla bitirdim.
