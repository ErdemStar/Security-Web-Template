Hoşgeldiniz

Bugün sizlere Tr0ll 1'in nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız hedef'e root olup /root/proof.txt'i okumak olacaktır.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile hedef'in ip adresini öğrenme
  • Nmap ile açık port'ların tespiti
  • Dirb ile 80 portu üzerinden bilgi toplama
  • İndirilen dosyalar içerinden strings ile metin arama
  • SSH Brute force ile kullanılabilir bir user bulma

İlk olarak hedefin kullandığı ip adresini bularak CTF'e başladım.

Ip adresini bulduktan sonra açık olan port'lar ve servisler hakkından bilgi edinmek için Nmap'ı kullandım.

Dirb ile 80 portu üzerinden çalışan web server hakkında bilgi toplamaya çalıştım.

Dirb'e ek olarak nikto kullanarak 80 port üzerinde bilgi toplamaya devam ettim.

Elde ettiğim bilgiler doğrultusunda artık web server üzerinde gezinmeye başladım.Fotoğrafda sizlere farklı path'lerde bulduğum fotoğrafları gösterdim.Bunların içersinde bir string olabileceğini düşünüp "strings" ile baktığımda sonuç alamadım.

Nmap çıktısından görebileceğiniz gibi hedef üzerinde FTP server eski kaldığı için herhangi bir doğrulama gerektirmeden tarayıcı üzerinden login oldum.Görebileceğiniz üzere .pcap çıktısıyla karşılaştım.Bunu incelemek için cihazıma indirdim.

Daha sonra strings komutu ile .pcap dosyası içersinde var olan metinleri görüntülemeye çalıştım.Burada "sup3rs3cr3tdirlol" yazısı gözüme çarptı.

Bu yazıyı URL üzerinde path olarak verdiğimde resimde gördüğünüz ekran ile karşılaştım.Burada da "roflmao" adlı dosyayı incelemek üzere indirdim.

İçindeki metinleri görmek için tekrar "strings" komutunu kullandığımda "0x0856BF" şöyle bir hexadecimal adres gözüme çarpı.

URL'e bu "0x0856BF" hexadecimal'i yazdığımda tekrar farklı bir path ile karşılaştım.Burada username ve password'lerin olduğu wordlist buldum.

Ekranda sizlere username ve password'lerin içeriklerini gösterdim.Burada password'lerin içersine dosya adı olan "Pass.txt" de ekledim.(Diğer türlü sonuç alamadım.)

Hydra ile SSH portuna Brute Force yaptığımda "overflow:Pass.txt" bilgileriyle login olduğumu gördüm.

Hedef sisteme SSH ile login olduktan sonra işime yarar şeyler aradım fakat pek birşey bulamadım.Daha sonra hedef sistemle ilişki bir exploit indirdim.

Exploit'i sistemde çalıştırdığımda root olduğumu gördüm.Daha sonra /root/proof.txt adlı flagi okuduktan sonra bir CTF'i daha bitirmiş oldum.
