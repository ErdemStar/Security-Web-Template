Bugün sizlere LAMPSecurity CTF4'ün nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız hedefe sızıp root olmaktır.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile IP adresi tespiti
  • Nmap ile açık port'ların tespiti
  • Dirb ile 80 portu üzerinde arama yapma
  • Nikto ile hedef hakkında bilgi elde etme
  • SQL Injection
  • Local File Inclusion

İlk adımda netdiscover komutu ile hedefin kulladığı IP adresini tespit ettim.

Nmap kullanarak hedef üzerinde çalışan servis ve portları buldum.

80 portu üzerinden gizlenmiş yada var olan yolları bulmak için dirb'i kullandım.

Dirb çıktısının tek başına yettiğini düşünmediğim için ve nikto ile hedefin 80 portu hakkında daha fazla bilgi edimeye devam ettim.

Dirb ve Nikto çıktılarından elde ettiğim Path'ler üzerinden gezinirken resimdeki şekilde bir URL ile karşılaştım.Böyle URL'ler görünce "'" işareti koymadan duramam :) Burada SQL Injection açığını keşfettim.

SQL Injection zafiyetinden sonra web üzerinde gezinmeye davam ettim.Daha sonra Null Byte LFI açığınında olduğunu keşfettim.

Daha sonra "dstevens:ilike2surf" bilgilerin kullanarak SSH login olmayı denediğimde sonuç aldım.Daha sonra "sudo -l" komutu ile hedef sistem üzerinde root yetkisiyle neleri çalıştıracağımı öğredim.Fotoğraftaki gibi bir çıktı elde edince doğrudan root olabileceğimi anladım.Bir CTF daha başaryıla bitti :)
