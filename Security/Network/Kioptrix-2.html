Hoşgeldiniz

Bugün sizlere Kioptrix Level 2'in nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız root olmak olacaktır.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile hedef'in ip adresini öğrenme
  • Nmap ile açık port'ların tespiti
  • SQL Login Bypass yapma
  • Hedefe shell olma

Hedefin kullandığı ip adresini netdiscoveri ile bularak CTF'imize giriş yapıyoruz.

Daha sonra nmap ile hedef üzerinden port/servis bilgisi elde etmeye çalıştım.

Hedefin ip adresini url'e yazdıktan sonra karşıma böyle bir sayfa geldi.Burada SQL Login Bypass yapmaya çalıştım.

Bypass işlemi başarıyla gerçekteştirdikten sonra fotoğrafda gördüğünüz gibi bir sayfa karşıladı beni.Bu sayfa üzerinde kullanıcıya ping atmak için bir input bırakılmış.

Bu input üzerinden bash komutları çalıştırabildiğimi fark ettim.Daha sonra bu input üzerinden reverse shell oluşturmaya çalıştım.Aşağıdaki fotoğrafta'da görebileceğiniz gibi bu ilk adımda hedefin "/tmp" yoluna bir "reverse.sh" adlı bir script oluşturuyorum ve bunun içerisinde reverse shell yapacak kodu yerleştiriyorum.Bir sonraki adımda bu script'in yetkilerine read , write ve execute ekliyorum.Son adımda bu script'i çalıştırıyorum.

Scripti çalıştırmadan önce kendi bilgisayarımda "netcat" kullanarak bir port açmıştım.Daha sonra script'i tetiklediğimde hedefe shell olduğumu görebilirsiniz.

Daha sonra hedef üzerinde araştırma yapmaya başladım ama işime yarayacak birşeyler bulamadım.Bende son umut olarak hedef işletim sistemi üzerinde çalıştırıp root olabileceğim bir exploit aradım.Daha sonra "9542.c" adlı bir exploit bulup çalıştırdığımda root oldum.Bu adımdan sonra CTF'i başarıyla tamamladım :)
