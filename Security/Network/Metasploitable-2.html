Hoşgeldiniz

Bugün sizlere Metasploitable 2'in nasıl çözüldüğünü göstermek istiyorum.Bu CTF'deki amacımız hedef üzerinde var olan birden fazla açıkları keşfetmektir.CTF hakkında daha fazla bilgiye ulaşmak için tıklayınız.

Geçtiğimiz adımlar

  • Netdiscover ile hedef'in ip adresini öğrenme
  • Nmap ile açık port'ların tespiti
  • Msfconsole ile birden fazla servis üzerinden shell alma
  • Hydra ile Brute force saldırıları

Hedefin ip adresini Netdiscover ile belirledikten sonra Nmap kullanarak hedef hakkında bilgi toplayarak hareket ettim.

Hedef üzerinde rastgele exploit ararken Msfconsole ile aşağıdaki fotoğrafdaki exploit'i kullanarak shell aldım.

Elimde bir wordlist olmasını için açık olan SMB portuna Enum4linux tool'u kullanarak username bilgisi almaya çalıştım.Daha sonra "cut" komutu kullanarak düzgün bir formata çevirdim.

Aramaya devam ederken Smb üzerinde var olan bir exploit'i Msfconsole'la keşfedip shell aldım.

Daha sonra 1099 portu üzerinde çalışan Java server'la alakalı bir exploit bulup shell aldım.

1524 portu ile alakalı birşey bulamadım daha sonra Telnet ile bağlanmayı denediğimde root olabildiğimi gördüm.

Aynı şekilde 3632 portu üzerinde çalışan server ile alakalı araştırma yaptığımda Msfconsole üzerinde bir exploit olduğunu keşfedip , Msfconsole'u kullanarak shell aldım.

8787 portu üzerinde çalışan server ile alakalı araştırma yaptığımda Msfconsole üzerinde bir exploit olduğunu keşfedip , Msfconsole'u kullanarak shell aldım.

Son olarak 58295 portu üzerinde bir başka Java serveri olduğunu öğrenip Msfconsole kullanıp shell aldım.

Daha öncelerde elde ettiğim wordlist'i Hydra ile FTP Brute Force yaparak elde ettiğim username-password'leri göstermek istedim.

Aynı şekilde Hydra ile SSH Brute Force yaparak elde ettiğim username-password'leri göstermek istedim.Bu adımdan sonra fazla abartmayarak CTF'i sonlandırmış oldum :)
